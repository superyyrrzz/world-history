# 历史地图

一个中文互动世界史小应用。从中国历史出发，通过共享时间轴观察同一时期的罗马、印度和希腊/拜占庭。

## 功能

- 可拖动的公元前 300 年至公元 589 年时间轴
- 基于 `vis-timeline` 的横向时间轴，支持拖动、缩放和横向滚动
- 中国列支持分道显示，用于呈现三国、十六国、南北朝等并立政权
- 文明筛选与关键年份快速跳转
- 当前年份动态信息卡，随滑块同步展示各文明状态
- 基于 `Floating UI` 的悬浮详情，自动避免跑出屏幕
- 支持桌面、平板和手机

## 本地运行

应用现在是一个标准 Node.js 静态应用。首次运行先安装依赖：

```bash
npm install
npm run dev
```

然后访问 `http://127.0.0.1:4173`。

## 内容说明

当前版本是教学原型，历史内容经过简化，以便儿童理解。它不应被视为专业历史资料。后续版本应为重要史实补充权威来源，并邀请历史教育工作者审阅。
